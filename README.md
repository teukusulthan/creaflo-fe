Creaflo is an AI-powered content assistant for creators, marketers, and small businesses.  
This repository contains the **frontend** app for Creaflo: the interface for generating hooks, captions, and hashtags from your ideas.

---

## Features

- Simple form to input topic/idea for content
- Generate multiple variations of:
  - Ideas
  - Hooks / scroll-stoppers
  - Captions
  - Hashtags
- Options for language
- Copy-to-clipboard and responsive UI

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui

---

## Getting Started

### 1. Clone & Install

```bash
git clone https://github.com/teukusulthan/creaflo-fe
cd creaflo-frontend
npm install
```

### 2. Environment Variables

```bash
NEXT_PUBLIC_API_URL= http://localhost:3000/api/v1
```
